# beMore Design Studio

> **Function First. Form Follows.**
> Architectural and Interior Design Practice based in Bengaluru, calibrating aesthetic intent with asset yield.

This is a Next.js application powering the core portfolio, detailed services matrix, and hybrid strategy diagnostics for **beMore Design Studio**.

---

## 🏗️ Technical Stack

-   **Frontend:** Next.js (App Router) & React hydration cycles.
-   **Styling System:** Vanilla CSS (`.module.css`) backed by Fluid Clamp sizes and offset gold framing.
-   **Components Layout:** Asymmetry-first Z-index framing replacing standard bento templates securely.

## 🛠️ Main Features

-   **Detailed Services Stack:** Staggered visuals backed by offset number typography offsets replacing default cards.
-   **Process narrative matrix:** Narrative frame offsets showcasing core Philosophy frameworks.
-   **Correct Responsive Sizing:** Recomputed header clamp scaling threshold frames allowing 12vw responsive buffer safely.

---

## 🚀 Getting Started

npm run dev
